# Explotacion de IA

Repositorio para organizar materiales de un articulo sobre explotacion de IA.

## Estructura

- `imagenes/`: recursos visuales del articulo.
- `textos/`: notas, borradores y contenido textual.
- `documentos/`: archivos de apoyo o documentos finales.
- `articulo/`: versiones del articulo.
- `script/`: scripts relacionados, incluido el contador en Python.
